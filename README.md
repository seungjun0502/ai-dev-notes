# 🤖 AI Dev Notes

AI에 대해 한발자국 나아가고 싶어 만든 학습 노트 입니다. 
LangChain, Vector DB, RAG, Agent, MCP 등 실전 개념을 정리합니다.

## 📚 주제별 노트

| 주제 | 한 줄 설명 |
|------|-----------|
| [LangChain](./langchain/) | LLM을 부품처럼 조립하는 프레임워크 |
| [Vector DB](./vector-db/) | 의미 기반 검색을 가능하게 하는 DB |
| [RAG](./rag/) | 외부 데이터로 LLM 답변 품질을 높이는 패턴 |
| [AI Agent](./agents/) | LLM이 스스로 도구를 선택하고 행동하는 패턴 |
| [MCP](./mcp/) | Claude에게 외부 서비스 접근 권한을 주는 프로토콜 |
